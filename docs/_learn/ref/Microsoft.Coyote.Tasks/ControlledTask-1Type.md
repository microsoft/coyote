# ControlledTask&lt;TResult&gt; class

Represents an asynchronous operation that can return a value. Each [`ControlledTask`](ControlledTask-1Type) is a thin wrapper over Task and each call simply invokes the wrapped task. During testing, a [`ControlledTask`](ControlledTaskType) is controlled by the runtime and systematically interleaved with other asynchronous operations to find bugs.
