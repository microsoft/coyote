# Configuration.WithPartialOrderSamplingEnabled method

Updates the configuration with partial-order sampling enabled or disabled. If this reduction strategy is enabled, then the runtime will attempt to reduce the schedule space by taking into account any 'READ' and 'WRITE' races declared by invoking [`Read`](../../Microsoft.Coyote.Runtime/SchedulingPoint/Read.md) and [`Write`](../../Microsoft.Coyote.Runtime/SchedulingPoint/Write.md).

```csharp
public Configuration WithPartialOrderSamplingEnabled(bool isEnabled = true)
```

| isEnabled | If true, then partial-order sampling is enabled. |
